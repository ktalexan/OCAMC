# General Formatting
- **Indentation:** Use 4 spaces for nested lists or LaTeX environments.
- **Spacing:** Use spaces around mathematical operators in equations (e.g., $a + b = c$).
- **Line Breaks:** Do not use hard line breaks for sentences; allow word wrap.

# Naming Conventions
- **LaTeX:** Use `snake_case` or `kebab-case` for labels and file references (e.g., `\label{fig:traffic_model_result}`).
- **Markdown:** Use `kebab-case` for image assets and link references.
- **Logic:** Use clear, structural names for sections and labels that reflect the document hierarchy.
